# Applied Behavior Analysis, Minor
## Required Courses 
The minor consists of the following required courses (18 hours). EDBA 3301 is a prerequisite for all other EDBA courses. Students who are minoring in ABA must earn a grade of C or better on all ABA minor courses (i.e., those starting with EDBA). 
Course List Code | Title | Credits  
---|---|---  
Introduction to Applied Behavior Analysis | 3  
Introduction to Research Methods in Applied Behavior Analysis | 3  
Ethical and Professional Behavior in Applied Behavior Analysis | 3  
Select an additional 9 hours of coursework from the following: | 9  
| Advanced Research Methods in Applied Behavior Analysis  
| Functional Behavior Assessment  
| Applied Behavior Analysis Applications for Behavior Change  
| Selecting and Implementing Applied Behavior Analysis Interventions  
| Personnel Supervision in Applied Behavior Analysis  
| Introduction to Autism Spectrum Disorder  
Total Credits | 18
